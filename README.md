# To Do Liste

Application de gestion de liste des tâches avec Webpack.

## Installation

```bash
npm install
```

## Commandes

- `npm start` : Lance le serveur de développement sur http://localhost:9000
- `npm run build` : Génère la version de production dans le dossier dist/
- `npm run dev` : Mode développement avec watch
